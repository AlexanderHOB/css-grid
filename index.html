CSS - GRID

CONCEPTOS FUNDAMENTALES

Grid Container

Es el elemento padre donde se asigna un display:grid

Grid Item

Hijos directos de un Grid Container

Grid Line

Lineas divisorias horizontales y verticales

Grid Track

Espacio entre dos lineas adyacentes, filas y columnas

Grid Cell

Espacio entre 2 filas adyacentes y 2 columnas adyacentes

Grid Area

Espacio que rodeado por 4 grid lines

Definiendo Columnas

Primero al grid container que es el padre le agragramos el display:grid

Luego para definir las columnas del container padre utilizamos la propiedad grid-template-columns: 25% 50% 25%
